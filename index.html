Software Development Institute

Command Result
add . git: Prepares untracked items to be commited
branch git: Creates a new version of the code for you to edit, with its own history.
cat Displays the file content in the CLI
checkout git: Switches to a specified branch of the code
checkout -b git: Creates a new branch and it becomes the active working branch
cd Changes the current directory
commit git: Adds a comment about the changes that were made.
ls Lists contents (the files) of the current working directory
Options Meaning
-l Displays the size of each item in bytes
-h Displays size of file with the suffix
-a Displays hidden files within a directory
mkdir Makes a new directory at the current location
merge git: Pulls copies into specified file to remove additional copies.
pwd Prints the absolute path of your current working directory
pull git: Adds contents from another file to the working file.
push git: Adds content from current working file to another file.
remote git: Used when referencing a remote directory
status git: checks the current working directory, lists changes in documents
touch Creates a new file in the current location
~/ References the user folder within the computer
../ References the directory before the current directory
